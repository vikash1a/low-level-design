### Encapsulation 
- It is a way to restricting access to class data.
- Example
  - class property is private and setter & getter is public

### Abstraction 
- It means abstracting the some details and exposing cetain details only.
- Example 
  - Abstract class has certain method implemented and certain are left to be overriden 

### Inheritance 
- A way to create different implemenation in form of a class using base structure (abstract class or interface)
- Example 
  - Class extends a abstract class
  - Class imlements a interface

### Polymorphism
- The abilty to take different form for same method based on the way it is called.
- Example
  - get(1), get(1,2) will use different implementation details as argument is different.
